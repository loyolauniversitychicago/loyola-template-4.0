Cards pattern

The card pattern is controlled by two stylesheets.

cards-columns.css

This stylesheet establishes the grid and stucture variations of the content.

Values of "grid-wrapper-min-max" can be edited to alter the column display.

.grid-wrapper-min-max {

grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); - changes column width.

grid-gap: 1.5rem; - changes gutter spacing.

}

cards.css

This stylesheet manages the standard typography and the variations of the typography per class. The classes to alter the presentation are "card", "card-horizontal", and "number". These are added to "grid-wrapper-min-max".

Customizable elements

• Container - Set width - add "container-100" class for full width or add "container-50" for narrow width

• Grid - Set minimum width of "grid-wrapper-min-max" - minmax(300px, 1fr)

• Content display - add "card" or "number" class to "grid-wrapper-min-max"

• Card orientation - vertical and horizontal - add "card" class or "card card-horizontal" classes to "grid-wrapper-min-max"

• Card background color - add "bg-gray-extra-light" or "terracotta-light" class

• Button color - add "maroon", "gold", or "patina" class
