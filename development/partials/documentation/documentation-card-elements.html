Follow best practice guidance. Word counts are strongly suggested.

  1. Image - One of two image sizes can be used. 1000px x 560px and 1000px x 1000px.
    For more information about images, see images documentation
  2. Label - Short label to add context, or the date of an event
  3. Title - Brief 5-8 word headline or title
  4. Identifier - Person or subject
  5. Explainer - Title or subline
  6. Readout - Brief 10-30 word paragraph.
  7. Link - Brief "Call to Action" - For example, READ MORE.
  8. Button - Brief "Call to Action" - For example, REGISTER NOW.
